\u00bfQuienes son los parientes inmediatos?<\/h3>\n

El t\u00e9rmino “parientes inmediatos” tiene un significado muy espec\u00edfico seg\u00fan las leyes de inmigraci\u00f3n de EE. UU. Se refiere al c\u00f3nyuge e hijos solteros menores de 21 a\u00f1os de un ciudadano legal de los Estados Unidos. Si el ciudadano es mayor de 21 a\u00f1os, sus padres tambi\u00e9n se consideran parientes inmediatos. La definici\u00f3n de “parientes inmediatos” es importante porque el gobierno de los Estados Unidos permite que un ciudadano estadounidense patrocine a sus parientes inmediatos para obtener visas de inmigrante para ingresar y permanecer en los Estados Unidos de forma permanente. A diferencia de otros tipos de visas familiares, las visas de pariente inmediato no se limitan a un cierto n\u00famero anual.<\/p>\n

Obtenci\u00f3n de una visa de familiar inmediata<\/h3>\n

Para obtener una visa de inmigrante de pariente inmediato para un pariente extranjero, se deben cumplir una serie de requisitos. Estos requisitos incluyen, pero no se limitan a:<\/p>\n

    \n
  • El ciudadano estadounidense patrocinador debe presentar una Petici\u00f3n para familiar extranjero (formulario I-130) completa en nombre de cada familiar que necesite una visa.<\/li>\n
  • Debe documentarse la relaci\u00f3n entre el extranjero y el ciudadano patrocinador. Dicha documentaci\u00f3n puede incluir registros de matrimonio y nacimiento.<\/li>\n
  • Si el pariente es un c\u00f3nyuge, el patrocinador debe demostrar que el matrimonio es de buena fe. Esto significa que el matrimonio no fue solo parte del esfuerzo por ingresar a los Estados Unidos.<\/li>\n
  • Si el matrimonio comenz\u00f3 hace menos de dos a\u00f1os, el c\u00f3nyuge extranjero solo es elegible para la residencia permanente condicional durante dos a\u00f1os. Se necesitar\u00e1 una presentaci\u00f3n adicional para levantar las condiciones si se demuestra que el matrimonio es de buena fe.<\/li>\n
  • El ciudadano estadounidense patrocinador generalmente deber\u00e1 presentar una declaraci\u00f3n jurada de apoyo para demostrar que el solicitante familiar no requerir\u00e1 asistencia p\u00fablica si se aprueba la solicitud.<\/li>\n<\/ul>\n

    El tr\u00e1mite de la solicitud depender\u00e1 de si el familiar extranjero se encuentra actualmente en Estados Unidos. Si el familiar ya est\u00e1 aqu\u00ed, se puede presentar una solicitud de Ajuste de Estatus<\/a> al mismo tiempo que el I-130. Si el familiar est\u00e1 fuera de los EE. UU., la solicitud se manejar\u00e1 a trav\u00e9s del procesamiento consular.<\/p>\n
